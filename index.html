Empowering women to drive change in the tech sector

We are a community of volunteers, mentors, and self-organised learners. We aim to close the gender gap in the tech sector by motivating and creating a safe space for women to develop coding skills at their own pace.

Learn to code

Learn by building projects in collaboration with others with a similar skill set. We help you join a self-organised learning circle guided by a mentor. We organise bi-annual intro events where we launch new learning circles and offer practical mentor-led workshops to help you install tools, code your first project and get started.

If you want to join a learning circle, you can sign up now to get notified via email about our next learning circle formation event.

Join a learning circle

Learning circle

When joining a learning circle, you commit to:

  • Choose a learning path & define your own learning goals
  • Build a project together & develop your portfolio
  • Keep each other motivated and on track
  • Meet up weekly with your learning circle at an agreed place and time

Mentorship

Learning circles are guided by mentors who:

  • Organise three intro workshops to help you get started
  • Help you to achieve your goals
  • Give technical feedback with code reviews
  • Meet up monthly with you for a mentorship session

Community

When you join a learning circle, you get access to our community on Discord where you can:

  • Ask questions
  • Connect to peers
  • Receive guidance from mentors
  • Network

Learning paths

You can take one of the following paths:

  • Introduction to programming with Python

    Python is a versatile, beginner friendly programming language you can use for data science, machine learning, app development and more.

  • Web Development with HTML, CSS & JavaScript

    HTML defines the structure of a web page and CSS defines its style. JavaScript is the programming language of the web.

  • Game Development with Unity

    If you want to learn the basics of game development, we recommend starting with Unity.

Mentor

We are looking for mentors who can guide the learning circles. If you have expertise in Python, web or game development and want to share your knowledge with a highly motivated and international community, please get in touch.
